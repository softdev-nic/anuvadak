
 
 Anuvadak
-----------------------------------------------------------------------------------------------------------------------------------------------------
 A simple and responsive translation web app built with React.
It connects to a custom backend API to translate text between multiple languages.

🚀 Features
✍️ Enter text and translate instantly
🌍 Select target language
🔄 Real-time API integration
⚡ Fast and minimal UI
📱 Responsive design
🧠 Tech Stack
React
JavaScript (ES6+)
Fetch API
Tailwind CSS
